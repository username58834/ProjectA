# ProjectA
Є бібліотека, яка має назву, списки працівників, членів бібліотеки (читачів), авторів та книг. Бібліотека може наймати та звільняти працівників, додавати та видаляти читачів. 

В кожної книжки є назва, жанр, автор, дата публікації, рейтинг, інформацію чиє вона позиченою та час, коли її було позичено. 

В працівників, читачів та авторів є ID, ім'я, прізвище та дата народження. 

Працівники мають посаду та зарплату. Працівники бібліотеки можуть видавати читачам книгу, приймати книгу, додавати та видаляти книжки та авторів. 

Читачі мають дату вступу до бібліотеки. Читачі можуть позичати та повертати книгу, залишати відгук. 

Автори мають країну походження та біографію.
